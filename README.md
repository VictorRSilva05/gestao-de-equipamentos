# Gestão De Equipamentos

![gif do programa](https://imgur.com/9MqocVD.gif)

## Descrição 

Júnior controla um estoque muito importante na sua empresa, porém ele realizava todo 
esse processo pelo Excel, o que tornava o processo muito lento e propenso a erros. 
Júnior então, entrou em contato com a Academia do Programador para que nós pudessemos
ajudá-lo a criar um programa que suavizasse sua rotina de trabalho.

## Design

O design do programa foi inspirado na empresa Neucom, do jogo Ace Combat 3 Electrosphere.
A paleta de cores foi escolhida para remeter a um ambiente tecnológico e futurista, remetendo
ao jogo.
A fonte utilizada foi a "Orbitron", que é uma fonte de estilo futurista, que se aproxima da 
usada no jogo.

## Como usar 

**Clone o repositório**

```
https://github.com/VictorRSilva05/gestao-de-equipamentos
```

**Navegue até a pasta raiz da solução**

```
cd GestaoDeEquipamentos
```

**Restaure as dependências**

```
dotnet restore
```

**Navegue até a pasta do projeto**

```
cd GestaoDeEquipamentos.ConsoleApp
```

**Execute o projeto**

```
dotnet run
```
